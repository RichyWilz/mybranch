NOTE: Computer languages are categorized under User friendly and User unfriendly.

USER FRIENDLY AND USER UNFRIENDLY LANGUAGES

User friendly languages are those that are easily understood by users when being used. This is because they are easy to relay messages to the user and vice-versa.eg
Message from computer if what you’re looking for isn’t there:
No Results found

User unfriendly languages are those that are hard to be understood by users when being used. This is because they are easy to relay messages to the user and vice-versa.eg
Message from computer if what you’re looking for isn’t there:
Output=0

Examples of User friendly languages

  • Python
  • Java-script
  • Swift
  • Scala
  • Go

Examples of User unfriendly languages

  • Haskell
  • C++
  • ASM
  • Prolog
  • Rust
